# Prodigy Infotech Backend Internship - Task 1

## Basic REST API using Node.js and Express

This project implements a simple REST API that performs CRUD operations for managing users.

## Technologies Used
- Node.js
- Express.js
- UUID
- Postman (for testing)

## API Endpoints

### Create User
POST /users

### Get All Users
GET /users

### Get Single User
GET /users/:id

### Update User
PUT /users/:id

### Delete User
DELETE /users/:id

## How to Run

1. Install dependencies
npm install

2. Run the server
node server.js

3. Test APIs using Postman
